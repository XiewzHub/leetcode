### 总结

- 每一个数都可以用和他等值的`1`相加，这样的结果只有一个
- 对于n来说，它最少可以有`n/2+1`个数字相加

- 每个数字可以使用的方法为：

  ```
  1:1
  2:1+1
  3:1+2
  4:1+3+1
  5:1+4+3
  6:1+5+5+1
  7:1+6+7+4
  8:1+7+9+7+1
  ...
  从上往下看，每一位相加的数都是递增的，是个等差数列
  ```

- 根据上面数据可以看出对于n，可以对应的结果可以由一下几个数字相加

  + 第0位 ：1+0  `1+0*（n-0）`

  + 第1位：1+1 加了n-1-1次  `1+1*（n-2）`

  + 第2位：1+2 加了n-3-1次`1+2*（n-4）`

  + 第3位：1+3 加了n-6   次 `1+3*（n-6）`
  + ... 依次类推
